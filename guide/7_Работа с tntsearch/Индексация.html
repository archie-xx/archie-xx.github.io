# Индексация

####AHTUNG!!! Есть ограничения по результатам выдачи, можно использовать для разработки, тестирования и CI. На проде лучше использовать ElasticSearch

Пример: App\Models\Traits\UserSearchable.php

  1. Создайте trait «ModelSearchable», где вместо Model подставьте название целевой модели

  2. Используйте в нем trait Searchable и реализуйте метод toSearchableArray

    use Laravel\Scout\Searchable;

    public function toSearchableArray()
    {
        return [
            'id' => $this->id,
            'name' => $this->name,
            'username' => $this->username,
            'email' => $this->email,
            'phone' => $this->phone,
            'role' => $this->role->name,
        ];
    }

  1. Укажите созданный trait для целевой модели

  2. Проиндексируйте модель php artisan scout:import App\\Models\\User
