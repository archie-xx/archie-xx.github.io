# Настройка страницы dashboard

Для создания информативной главной страницы админки (Dashboard) могут использоваться графики и прочие плюшки.

# Добавление элементов на страницу

Для того чтобы изменить список элементов, выводимых на страницу, необходимо перейти в файл dashboard.js. В данном файле по дефолту экспортируется объект модели дашбоарда, который имеет следующую структуру:

export default {
  title: 'Dashboard',

  rows: [
    {
      classList: 'mb-3',
      items: [
        {
          type: ICON_CARD,
          title: 'Всего пользователей',
          variant: 'primary',
          icon: 'user',
          value: dashboardData => dashboardData.total_users,
          linkTo: '/users'
        }
      ]
    },

    {
      title: 'Новые пользователи',
      makeCard: true,
      items: [
        {
          type: LINE_CHART,
          value: dashboardData => {
            const dataArr = dashboardData.daily_new_users_statistic

            return {
              labels: dataArr.map(item => item.date),
              datasets: [
                {
                  label: 'Новых пользователей',
                  backgroundColor: '#ffbed7',
                  data: dataArr.map(item => item.amount)
                }
              ]
            }
          }
        }
      ]
    }
  ]
}

ВАЖНО

Посмотреть структуру данных для графиков можно на сайте плагина: https://www.chartjs.org/docs/latest/

В данном объекте можно задать Название страницы (то, что будет отображаться сверху при заходе). Каждый элемент массива rows является отдельно выводимой карточкой. Каждая карточка может иметь один или более элементов. Чтобы добавить элемент на карточку, необходимо добавить в её массив items новый объект. Добавленный объект обязательно должен иметь поле type, который отвечает за тип выводимого графика.

# Доступные типы графиков

Импорт из файла constants/dashboard/element-types

export const ICON_CARD = IconCard
export const LINE_CHART = LineChart
export const PIE_CHART = PieChart
export const BAR_CHART = BarChart
export const HORIZONTAL_BAR_CHART = HorizontalBarChart

# Кастомизация графика

Помимо этого поля необходимо указать все необходимые данные для графика. Например, для графика icon-card такими полями будут title, variant, icon.

# Значения графиков

Для присвоения значения графику необходимо реализовать метод value. Например, реализация для графика spline-chart: В параметры данного метода передается объект dashboardData, который содержит данный полученные с сервера (../admin/dashboard)

# Перенаправление при нажатии

Для того чтобы при нажатии на график проиходило перенаправление на определенную страницу, необходимо добавить поле linkTo:

c = {
  type: ICON_CARD,
  title: 'Всего пользователей',
  variant: 'primary',
  icon: 'user',
  linkTo: '/users',
  value: dashboardData => dashboardData.users
}
