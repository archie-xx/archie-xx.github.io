# Кастомизация страницы представления редактирования

# Слоты

В CRUD/Form.vue добавлены слоты, которые позволяют добавить свои элементы, без переопределения всей страницы.

# topHeader

Этот слот поможет переопределить заголовок табличной страницы

<slot name="topHeader">

Использование:

<div slot="topHeader">
    <strong>{{title}}</strong>
</div>

# header

C помощью этого слота можно добавить какую-либо кнопку после заголовка:

 <slot name="header"/>

Использование:

<div slot="header">
    <strong>{{title}}</strong>
</div>

# footer

Слот для добавления чего-нибудь кастомного до кнопки сохранения:

<slot name="footer"></slot>

# saveButton

Слот, чтобы заменить кнопку сохранения:

<slot name="saveButton"></slot>

# after_form

Слот после формы:

<slot name="after_form"></slot>

Слот можно переопределить на странице Form:

<template>
 <div>
   <Form :model="model">
     <!-- ^ можно вешать свои методы на @событие="метод" и  :параметр="поле" -->
     <template slot="header">qwe</template>
     <!-- где slot="name слота" -->
   </Form>
 </div>
</template>

<script>
import Form from "../../../components/CRUD/Form";
import model from "../../../models/Example";
export default {
 name: "ExampleForm",
 data() {
   return {
     model
   };
 },
 components: {
   Form
 }
};
</script>

# Переопределение страницы формы (Form.vue)

Если необходимо реализовать супер кастомные поля, которые не могут быть реализованы дефолтными инструментами, можно просто переопределить файл Form.vue:


<template>
  <div>
  <!-- code -->
  </div>

 <Form :model="model">
      
 </Form>
</template>

<script>
import model from "../../../models/Role";
import Form from "../../../components/CRUD/Form";
export default {
  name: "NameForm",
  data() {
    return {
      model
    };
  }
};
</script>
