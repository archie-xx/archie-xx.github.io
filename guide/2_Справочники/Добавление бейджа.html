# Добавление бейджа

# Backend

Для того чтобы реализовать бейджи на бэке

Необходимо

  1. В BadgeController реализовать функцию подсчета необходимых данных, которая возвращает кол-во
  2. В методе BadgeController@counter добавить в объект ключ в значении которого вызывается функция из п.1

# Frontend

В модель добавить объект вида

export default {
  ...
  label: {
    color: 'success',
    badge: 'admin_user_all_count'
  }
}

где badge это ключ возвращаем из бэкенда, а color это css class цвета
